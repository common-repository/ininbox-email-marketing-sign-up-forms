== Description ==

<p>The Official INinbox Sign Up Form plugin makes it easy to grow your subscriber email lists! Use this plugin to implement your email newsletter signup forms into your WordPress site. If you'd like to learn more about ININbox, signup for a <a href="http://www.ininbox.com">FREE account</a> and explore.</p>
<p>Once the plugin is activated, you'll be able to preview, select and insert any of your INinbox webforms. Integrate them right into your website by choosing from one the 3 options: a widget, shortcode, or template tag. The setup takes jsut a few seconds. Simply enter your API key (which can be found in your <a href="http://www.ininbox.com/api">INinbox account</a>) in the settings and you're all set.</p>

<p>Official INinbox Forms Plugin Features:</p>
<ul>
<li>Automatically insert signup forms which send subscribers to an email list of your choice</li>
<li>Add unlimited forms by using the Widget, Shortcode or Template Tag</li>
<li>Quick links to preview your forms in INinbox</li>
<li>Quick links to edit your forms in INinbox</li>
<li>Quick links to create new forms in INinbox</li>
</ul>


== Installation ==
<ol>
<li>Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer</li>
<li>Go to the Plugins >> Installed Plugins Section and activate the plugin</li>
<li>Go to the INinbox Settings (under Settings > INinbox Settings)</li>
<li>Simply enter your API key (which can be found in your <a href="http://www.ininbox.com/api">INinbox account</a>). </li>
<li>Click Save Settings</li>
</ol>

== Frequently Asked Questions ==

<h4>What is INInbox?</h4>

<p>A complete, easy to use, email marketing solution. </p>

<h4>Do I need an INinbox account to use this plugin?</h4>

<p>Yes, you need an <a href="https://www.ininbox.com/free-sign-up.html">INinbox account</a> to use this plugin</p>

<h4>Is there a widget?</h4>

<p>Yes. Go to Appearance >> Widgets in your  WordPress Dashboard and look for the widget called "INinbox Form". Drag it into the widget area of your choice. You can then add a title, description and choose a form!</p>

<h4>Is there a shortcode?</h4>

<p>Absolutely! You can add any form to any post or page by pasting a shortcode (ex. [ininbox id="81" type="html"]). You can find all shortcodes on the INinbox Settings page.</p>

<h4>Is there a template tag?</h4>

<p>Of course! You can add a template tag into any WordPress file, example: <?php ininbox_form( '81', 'html' ); ?> You can find all template tags below the shortcodes on the INinbox Settings page.</p>

<h4>Where can I find the API Key?</h4>
<p>You can find your API key in your INinbox Account. For more details see: http://INinbox.com/api</p>

<h4>Is there support available?</h4>
<p>Not sure about one of the features or need more documentation? Get free support <a href="https://www.ininbox.com/support">here</a></p>
